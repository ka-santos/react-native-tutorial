# Tópicos Importantes para a primeira avaliação

- O que é React? é uma biblioteca para desenvolvimento de aplicações web e mobile

- O que é o React Native? é uma biblioteca para desenvolvimento de aplicações mobile

- O que é o Expo? é um framework para desenvolvimento de aplicações mobile

- Qual linguagem de programação utilizada na disiplina? javascript

- O que é o Node? um ambiente de execução do javascript

- O que é o npm? um gerenciador de pacotes do node(node package manager)

- O que é Framework e qual a diferenção para uma biblioteca? é um conjunto de recursos/ferramentas que alguém deixou preparado para resolver um problema tanto para front-end quanto para back-end (Exemplo: o React é um framework para front-end e o Node é um framework para back-end) "fornecendo uma estrutura/opinião para resolver o problema sendo mais flexivel ou não". 
já a biblioteca não fornece uma estrutura para resolver o problema, mas sim uma ferramenta para resolver o problema
(Exemplo: o React é uma biblioteca para front-end e o Node é uma biblioteca para back-end) não fornace uma opinião de como resolver o problema".
